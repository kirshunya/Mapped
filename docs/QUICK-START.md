This Visio file was created using a **custom Go program**:

```go

3. Generates: XML shapes with positions/colors
4. Creates: Final VSDX file
```

The program automatically:
- Converts CSV data to Visio shapes
- Applies color coding by layer
- Positions blocks with exact coordinates
- Maintains VSDX file format
